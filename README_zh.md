# MuseDashInfo+

[English](README.md) | [中文](README_zh.md)

### 概述
MuseDashInfo+ 是一个轻量级的 MuseDash 游戏模组，用于显示额外的游戏信息。本模组受 MuseDashCustomPlay 启发。

### 功能特性
- **击中/漏击计数器**：在游戏界面左上角实时显示击中与错过统计
- **谱面信息**：在游戏界面右上角显示当前歌曲名称与铺面难度等级
